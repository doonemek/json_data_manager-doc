# 拡張予定 (Roadmap)

## 短期目標
- **バリデーション機能**: 保存された分割ファイル間で ID の重複や欠損がないかを検証する。

## 中期目標
- **データ分析エンジン**:
    - 各 ID 範囲ごとのデータ密度可視化。
    - タグの使用頻度、最新更新日時の集計。
- **検索インデックスの構築**: メモリ上に `name` から ID を引くためのインデックスを保持し、検索を高速化。
- **処理実行時にデータの削除 or アーカイブ化**: outputやlogに残ったデータを削除 or アーカイブ化

## 長期目標
- **CLI インターフェースの強化**: 引数によって「マージモード」「分割モード」「分析モード」を切り替え可能にする。
